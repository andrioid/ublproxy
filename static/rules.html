Syntax reference
Block a domain: ||ads.example.com^
Block a URL pattern: /ads/* or ||example.com/ads/
Hide an element: ##.ad-banner or example.com##.sidebar-ad
Exception (allow): @@||example.com^ (overrides blocks, enables passthrough)
Unhide an element: #@#.ad-banner (overrides element hiding)

You can also use the element picker (Alt+Shift+B) on any proxied page to visually select elements to hide.
